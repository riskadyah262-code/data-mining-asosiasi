# Data Mining Asosiasi - Market Basket Analysis

## 📌 Deskripsi
Proyek ini merupakan implementasi Data Mining menggunakan metode **Association Rule Mining** (Analisis Keranjang Belanja / Market Basket Analysis). Tujuannya adalah menemukan pola hubungan antar produk yang sering dibeli bersamaan oleh pelanggan.

## 🎯 Tujuan
- Menemukan itemset yang sering muncul bersamaan dalam transaksi
- Menghasilkan aturan asosiasi antar produk
- Membantu pengambilan keputusan bisnis seperti penempatan produk dan strategi promosi

## 📂 Dataset
- **File:** `Market_Basket_Optimisation.csv`
- Dataset berisi data transaksi pembelian pelanggan di supermarket

## 🛠️ Tools & Library
- Python
- Jupyter Notebook
- Pandas
- MLxtend (Apriori & Association Rules)

## 🚀 Cara Menjalankan
1. Clone repository ini:
git clone https://github.com/riskadyah262-code/data-mining-asosiasi.git
2. Buka file `market_basket.ipynb` di Jupyter Notebook
